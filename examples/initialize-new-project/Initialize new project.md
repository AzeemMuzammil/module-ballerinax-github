# Initialize a New GitHub Project

This example demonstrates how to create a new repository on GitHub, initialize it with a README file, and add collaborators to the repository.

## Prerequisites

### 1. Set up
Refer to the set up guide in [ReadMe](../../../README.md) for necessary credentials.

### 2. Configuration

Configure GitHub API credentials in Config.toml in the example directory:

```toml
authToken="<GITHUB_PAT>"
```

## Run the Example

Execute the following command to run the example:

```bash
bal run
```
